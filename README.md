# Linux Production Server Project

This project demonstrates basic Linux server administration using Amazon Linux in AWS CloudShell.

## Skills Practiced

- Linux user and group management
- File permissions and ownership
- Networking commands
- Web server deployment
- Log monitoring
- Backup automation

## Technologies

Linux (Amazon Linux)
Nginx Web Server
Bash scripting

## Steps Performed

1. Created users and groups
2. Managed file permissions
3. Configured networking commands
4. Installed and ran Nginx web server
5. Monitored system and web server logs
6. Created automated backup script using tar

## Backup Script Example
#!/bin/bash
tar -czvf backup.tar.gz /project


## Outcome

Built and managed a Linux server environment including web hosting and automated backup processes.
